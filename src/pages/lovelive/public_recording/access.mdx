# 徒歩：ルートA <span id="walkA" />

牛久駅から会場までは徒歩25分ほどです。
Google map で「牛久市中央生涯学習センター」までの道を検索すると遠回りする道が出てくるので、<span style="color: red;">経路検索を使う場合は「牛久市立中央図書館」を目的地に</span>設定することをオススメします。

ルートAでは、アニメに登場した[牛久シャトー](/spot_note/chateau)と鬼塚姉妹の横断幕がある[市役所](/spot_note/city_hall)）を経由して会場に向かうルートを紹介します。

### スクランブル交差点まで
牛久駅から出発です。
改札を出て左に曲がってまっすぐ進み、[スクランブル交差点](/spot_note/scrumble)を斜め左に進んでいきます。

# 徒歩：ルートB <span id="walkA" />
牛久市ヘルスロード「シャトー通り散策コース」は途中で中央生涯学習センター前を通るため、こちらを通って会場に向かうこともできます。
所要時間はルートAとほぼ同じで25分ほどです。
ウォーキングコースには銅像が展示されており、シャトー・市役所を既に訪問済みの方にはオススメのルートです。

詳細は[シャトー通り散策コース](/course/chateau_walk)のページをご覧ください。

# バス <span id="bus" />
牛久駅からかっぱ号に乗る場合、以下のルートがあります。

- 運動公園ルート　中央生涯学習センター　すぐ
- 小坂団地ルート・右回り 東洋大牛久高校入口 徒歩5分

小坂団地ルート・右回りに乗る場合、「つくばセントラル病院前」まで乗車して[柏田神社](/spot_note/kashiwada_shrine)（アニメ3期11話に登場）に参拝していくのもオススメです。
